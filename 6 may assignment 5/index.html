Q1: What is HTML?

HTML stands for HyperText Markup Language. It's used to structure web pages.

Q2: What is CSS?

CSS stands for Cascading Style Sheets. It is used to style HTML elements.

Q3: What is JavaScript?

JavaScript is a scripting language used to create dynamic website content.

Q4: What is a Website?

A website is a collection of web pages accessible through the internet.

Q5: What is a Web Browser?

A web browser is software used to view websites like Chrome, Firefox, etc.
